# How to Use The GreenZone

The GreenZone exists to help the FC turn dungeon loot and spare gear into the maximum possible
Free Company Credits. When 25 or fewer members are online, hand-ins generate the GreenZone 3x FC
Credit bonus. The plugin helps members see when that bonus is active and how the FC is progressing
towards the weekly target.

## What Members Should Do

1. Open The GreenZone with `/fcmetrics`.
2. Open the Free Company window once so the plugin can read the current FC Credits and online count.
3. Use `/fcmetrics overlay` if you want the smaller overlay while playing.
4. Wait for the GreenZone message before handing in loot when possible.
5. Hand in gear to earn seals for yourself and FC Credits for the FC.

## Getting the Most Out of It

GreenZone is active when 25 or fewer FC members are online. Handing in loot during GreenZone gives
the FC 3x the normal FC Credits, while you still receive your seals.

If the plugin says GreenZone is inactive, consider waiting until fewer members are online before
handing in large amounts of loot. Small hand-ins are fine whenever you need seals, but big batches
are best saved for GreenZone.

## Weekly FC Target

Everyone uses the same shared weekly base and target. The FC goal is to earn the target amount of
FC Credits beyond that shared base during the challenge week.

The overlay week number is calculated automatically, while the shared base and target come from
the live `fc-week.json` file maintained by the FC.

## Personal Reward Tiers

Only your highest completed tier is paid each week.

```text
Tier 1: 500,000 seals = 1,000,000 Gil
Tier 2: 1,000,000 seals = 3,000,000 Gil
Tier 3: 1,500,000 seals = 4,000,000 Gil
Tier 4: 2,500,000 seals = 6,000,000 Gil
Tier 5: 3,500,000 seals = 10,000,000 Gil
```

Rewards are posted on Monday night. Please make sure Gwen is in your friends list.

## Useful Commands

```text
/fcmetrics
/fcmetrics overlay
/fcmetrics clear
```

Use `clear` only if you need to reset local tracking for the current character.
